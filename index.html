Firebase 설정

Firebase Realtime Database를 연결하면 모든 접속자가 동일한 데이터를 실시간으로 공유합니다.
완전 무료 (Spark 플랜, 1GB 저장공간)이며 설정은 최초 1회만 필요합니다.

📋 설정 방법 (5분)

  1. console.firebase.google.com → 새 프로젝트 생성
  2. 왼쪽 메뉴 Realtime Database → 데이터베이스 만들기 → 테스트 모드 선택
  3. 생성된 URL 복사 (예: https://xxx-rtdb.firebaseio.com) → 위 첫 번째 입력칸에 붙여넣기
  4. 프로젝트 설정(⚙) → 일반 탭 → 웹 API 키 복사 → 두 번째 입력칸에 붙여넣기
  5. 저장 & 연결 클릭 — 끝!
⚠️ Firebase가 연결되지 않았습니다. 현재 이 기기에만 저장됩니다. 지금 설정하기 →
